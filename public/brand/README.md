# BTC by Aradhana – Brand Assets

Add your logos and icons here from the **BTC branding kit**:

**Download from Google Drive:**  
[https://drive.google.com/drive/folders/1qaTQdMC868OEjcATKMzBEEqrLHdLbRUz](https://drive.google.com/drive/folders/1qaTQdMC868OEjcATKMzBEEqrLHdLbRUz)

## What to add

1. **Logo (header & footer)**  
   - From the **RGB** folder (for screen/web).  
   - Save as: `logo.png` or `logo.svg` in this folder (`public/brand/`).  
   - The site will use it automatically in the header and footer.

2. **Favicon / app icon**  
   - From the **ICONS** folder.  
   - Save as: `favicon.ico` in this folder, **or** put `icon.ico` in the `app/` folder for the browser tab icon.

3. **Other assets**  
   - You can add more files (e.g. from **CMYK**, **RGB**, **ICONS**) here and reference them in the code as `/brand/your-file.png`.

## Folder structure after adding files

```
public/
  brand/
    logo.png     (or logo.svg)  ← used in header & footer
    favicon.ico  ← optional; or use app/icon.ico
    README.md    ← this file
```

Until you add `logo.png` or `logo.svg`, the site will show the text “Be The Change” as the logo.
